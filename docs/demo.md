# demo: 60 seconds with ro

a tour for someone seeing it for the first time. read aloud, hit each
beat. it's tight on purpose.

---

(0:00) "this is ro. it's my agent. one user, runs on my laptop, asks
before doing anything that touches another person."

(0:08) open `/overview`. point to the live trace card.

"the right side is what ro is doing right now. it's drafting a reply
to a recruiter, and it's about to ask me to approve it."

(0:18) hit cmd-k. type "what's on my plate today".

"it pulls from my profile, my calendar, my open prs, and my pending
approvals. nothing is hardcoded."

(0:30) click into `/memory`. show the profile editor on the left.

"this file is the source of truth. when i tell it 'i prefer 25 minute
meetings', it lands here. it gets injected into every conversation."

(0:42) click into `/inbox`. hover the pending approval card at the top.

"every reply waits for me. i can approve, edit, or reject in one click.
nothing goes out unless i say so."

(0:52) click into `/code`. point to the rohflow card.

"and when i want to actually code, it hands me off to claude code with
context already loaded."

(0:60) "that's it. one agent. no copy paste between fifteen tabs."
